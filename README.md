# Momna Zaheer Portfolio

Premium Next.js portfolio for a professional UI/UX and Graphic Designer.

## Stack

- Next.js (App Router) + TypeScript
- TailwindCSS
- Framer Motion
- React Three Fiber / Drei
- ShadCN-style UI primitives

## Run locally

```bash
npm install
npm run dev
```

## Structure

- `app/` routes and layouts
- `components/` reusable UI and page sections
- `data/projects.ts` centralized portfolio/case-study data
- `lib/` utility functions
